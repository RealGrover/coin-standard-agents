# Coin Standard Twitter Visual - Design Specifications

## Format: 1:1 (Square) - Best for Twitter

### Color Palette:
- Primary: #F7931A (Bitcoin orange)
- Secondary: #1A1A1A (dark background)
- Accent: #FFFFFF (text)
- Highlight: #00D4FF (accent blue)

### Typography:
- Header: Inter Bold, 48px
- Body: Inter Regular, 24px
- Small: Inter Regular, 18px

### Layout Options:

**Option 1: Split Screen**
- Left: "Before" with confused emoji
- Right: "After" with rocket emoji
- Center divider: → (arrow)

**Option 2: Top/Bottom**
- Top: Current Bitcoin UX problem
- Bottom: Coin Standard solution
- Divider: Bold line with "NO MORE DECIMALS"

**Option 3: Infographic**
- Circular flow: 100,000,000¢ → 1 ₿
- Percentage breakdown: 1¢, 1,000¢, 1,000,000¢, 100,000,000¢
- Final: "Intuitive. Familiar. No decimals."

### Key Elements:
1. **Headline:** "Bitcoin's Decimal Hell is Over"
2. **Visual:** Clear before/after comparison
3. **Math:** Simple 100,000,000¢ = 1 ₿
4. **Problem:** "0.00234 BTC" vs "234,000¢"
5. **Solution:** "No decimals. Just coins."
6. **CTA:** "Join the movement: coinpill.me"

### Hashtags:
- #Bitcoin
- #CoinStandard
- #UX
- #Crypto
- #BitcoinAdoption

### File Specs:
- Format: PNG or JPG
- Resolution: 1200x1200px (minimum)
- File size: Under 5MB
- Safe for work: Yes

### Brand Guidelines:
- Use Bitcoin orange prominently
- Keep text minimal and bold
- Use emojis sparingly but effectively
- Maintain professional but approachable tone
