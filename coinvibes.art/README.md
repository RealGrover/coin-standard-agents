# Coin Vibes

Artistic exploration of Bitcoin's Coin Standard UI — where 1 BTC = 100,000,000¢

## About

Coin Vibes is a creative project that showcases how the Coin Standard makes Bitcoin feel like real money. Through artistic UI examples, we demonstrate that:

- **¢ is intuitive** — Everyone understands cents
- **No more decimals** — Clean integer displays
- **Defeats unit bias** — 100M coins feels attainable vs 1 BTC

## Features

- **Interactive coin grid** — Click coins to see floating ¢ effects
- **UI examples** — Wallet displays, payment requests, invoices, mobile apps
- **Vibrant design** — Gradient accents and smooth animations
- **Responsive** — Works on desktop and mobile
- **Smooth scrolling** — Navigation with scroll effects
- **Parallax effects** — Hero section with depth

## How to Use

1. Open `index.html` in your browser
2. Explore the examples
3. Click on coins for interactive effects
4. Use the navigation to jump between sections

## Technical Details

- **Design System:** CSS custom properties for theming
- **Animations:** CSS transitions and keyframes
- **Interactivity:** Vanilla JavaScript (no frameworks)
- **Responsive:** Mobile-first approach
- **Performance:** Optimized for smooth 60fps animations
